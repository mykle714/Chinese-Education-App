## 6. Code changes made during the run

Three defects surfaced only under real execution.

### Fix 1 вЂ” `ANTHROPIC_API_KEY` guard blocked oracle mode
Every AI script has its own copy of the "ANTHROPIC_API_KEY not set" bail (**15
copies**), which fired before oracle mode could take over. Fixed at the single seam
(`run-log.js`) rather than editing 15 files. The key is **clobbered**, not merely
filled in when missing: oracle mode must make real credentials unreachable, so any
code path that bypasses the `messages.create` wrapper fails 401 instead of silently
spending.

### Fix 2 вЂ” chained-call scripts could never complete
`oracleCreate` treated an unknown promptId during apply as a fatal error. But several
scripts chain calls per row (definitions-array's pass-1в†’critic, long-definitions'
generatorв†’validatorв†’retry ladder, example-sentences' generatorв†’reviewer), and export
can only ever capture the **first** call because capturing unwinds the row. Apply
therefore always died on the second link. Now an unknown prompt during apply is
**captured and unwound** exactly as in export, making the phases a converging loop:
apply в†’ author the newly surfaced prompt в†’ apply again, one link per pass, until the
row completes. All three multi-call scripts completed this way (2, 2, and 2 passes).

### Fix 3 вЂ” flat-usage gate (В§6 of the skill)
Added at the user's request: if a round that wrote rows does not raise
`five_hour.utilization`, the answers came from outside this session вЂ” halt and report
rather than continue.

### вљ  Finding 2 вЂ” export accounting misses hand-rolled loops
`runner.js` counts `OracleExportSignal` separately ("рџ“¤ Oracle Export Complete!"), but
`backfill-process-definitions-array`, `backfill-long-definitions`, and
`backfill-example-sentences` each have their own loop and report captures as
`FAILED / Errors: 1`. Cosmetic вЂ” the prompt is captured and no DB write occurs вЂ” but
it makes a healthy export look like a failure. Fix: have those loops check
`err.oracleExport`, or migrate them to the shared runner.
